Thank you for downloading Leaf-Tailed Gecko.

This is a realistic 3d model and animations of a Leaf-Tailed Gecko. 
-Models folder contains LeaftaeledgeckoModelLowPoly.FBX, LeaftaeledgeckoModelMiddlePoly.FBX and LeaftaeledgeckoModelTowLODs.FBX.

-Animations folder contains FBX files of animations.
Walk and run animations are compatible with root motion.
-Animations/InPlace folder contains In Place variations of walk and run animations.
These are not used in the demo.
-Textures folder contains textures.
These are main contents of this package.

Demo folder contains an example use of the Bear model. 
-Demo/Scenes folder contains demo scene.
-Demo/Prefabs folder contains prefabs that are used in the demo scene.
-Demo/Scripts folder contains scripts that are used in the demo scene.
-Demo/Animator folder contains animator controller that is used the prefab.
